========================================
  Football Analysis 使用说明
  足球数据分析与预测系统 v1.0
========================================

一、系统要求
----------------------------------------
- Windows 10/11 (64位)
- 首次安装时自动下载并安装 Node.js（用于 Playwright 浏览器采集）
- 无需手动安装 Python - 安装程序已包含所有依赖

二、安装步骤
----------------------------------------
1. 双击运行 FootballAnalysisSetup.exe
2. 点击下一步，安装到默认位置
3. 安装完成后会自动：
   - 解压应用程序
   - 检测/安装 Node.js（如需）
   - 安装 Playwright Chromium 浏览器
   - 创建桌面快捷方式和开始菜单快捷方式

三、首次配置
----------------------------------------
1. 双击桌面图标 Football Analysis 启动服务
2. 浏览器会自动打开 http://127.0.0.1:5050/
3. 点击导航栏 配置 或访问 http://127.0.0.1:5050/config 填写 API 密钥

   必需配置：
   - OPENAI_BASE_URL       : OpenAI 兼容 API 地址
   - OPENAI_API_KEY        : OpenAI API 密钥
   - OPENAI_MODEL_RESEARCH : 模型名称（如 gpt-4o）

   数据采集配置：
   - COLLECTION_BASE_URL   : 采集用 API 地址
   - COLLECTION_APIKEY     : 采集用 API 密钥
   - COLLECTION_MODEL      : 采集用模型名称

   可选配置：
   - TAVILY_API_KEY        : AnySearch 备用采集
   - FOOTBALL_API_KEY      : 足球数据 API

四、主要功能
----------------------------------------

4.1 数据同步与采集
   - 首页点击 同步对赛 获取最新胜负彩期号和对阵
   - 点击 批量采集 自动采集各项数据（赔率、实力、身价、阵容等）
   - 数据来源：500.com 为主源，Playwright/AnySearch 为补充

4.2 AI 预测
   - 数据采集完成后点击 生成预测
   - 系统自动运行特征工程、概率建模、LLM 复核、目标批量策略
   - 预测结果包含：推荐动作（主推/轻仓/观望）、建议仓位、有效推荐

4.3 学习闭环
   - 在 学习闭环 区域设置学习窗口（如最近 100 期）
   - 点击 训练学习候选 进行离线策略搜索
   - 点击 回放补齐历史期 自动回溯更早期号补齐学习样本
   - 当前策略 #43：让球盘桶表策略，目标命中率 >= 70%

4.4 赛果反馈
   - 比赛结束后，在首页点击 同步赛果
   - 系统自动结算并更新 TOP3 精选命中率

4.5 TOP3 精选
   - 首页展示本期精选 3 场比赛
   - 已结算场次显示命中/错误、实际赛果和比分

五、完整操作流程
----------------------------------------
  1. 同步对赛 -> 2. 批量采集 -> 3. 生成预测 -> 4. 同步赛果 -> 5. 记录反馈

六、停止服务
----------------------------------------
- 关闭浏览器标签页不够，需停止后台进程
- 在应用首页点击 停止服务 按钮
- 或在命令行运行：taskkill /F /IM FootballAnalysis.exe

七、常见问题
----------------------------------------

Q: 页面打不开？
A: 确认服务正在运行。检查任务管理器中是否有 FootballAnalysis.exe 进程。

Q: 数据采集失败？
A: 检查网络连接。500.com 页面可能临时不可用。查看 采集质量 页面确认哪些字段采集失败。

Q: 预测结果为空？
A: 必须先完成数据采集。检查 analyses 表中是否有有效数据。

Q: 端口 5050 被占用？
A: 打开另一个命令行，运行：
    netstat -ano | findstr :5050
    taskkill /PID <进程号> /F

Q: 如何更新？
A: 运行新版 FootballAnalysisSetup.exe，安装程序会自动备份 .env 和数据库文件。

八、文件说明
----------------------------------------
安装目录：%LOCALAPPDATA%\\FootballAnalysis\\app
  FootballAnalysis.exe  - 主程序（已打包所有 Python 依赖）
  .env                  - API 配置文件（首次运行后自动生成）
  .env.example          - 配置模板（含所有可选字段说明）
  _internal/            - 内部资源（模板、静态文件、数据库）
    football_data.db    - 主数据库（SQLite，勿手动编辑）
    football_data_live.db - 实时数据缓存
    templates/          - Web 页面模板
    static/             - CSS/JS 资源

九、技术支持
----------------------------------------
- 查看日志：%LOCALAPPDATA%\\FootballAnalysis\\app\\*.log
- 备份恢复：安装目录的 backup_YYYYMMDD_HHMMSS 文件夹
- 配置文件：.env 文件（文本编辑器即可修改）
